Please install here LKH algorithm.
1. Get source code from http://www.akira.ruc.dk/~keld/research/LKH/ 
2. Build it (instructions available on page mentioned above).
3. Name executable "lkh.exe" (yes, .exe even under Linux) and put it into folder /algo/lkh (where this readme.txt is).
